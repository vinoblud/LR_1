1. Що таке Git? Git - це система перетворення  версій файлів
2. В якому статусі можуть перебувати файли при роботі з Git? "untracked file" або "master"
3. Для чого використовується  команда add? Команда git add додає вміст робочого каталогу в індекс для наступного комміту.
4. Для чого використовується  команда status?  Для перевірки, чи відстежуються файли чи ні
5. Для чого використовується  команда commit? Команда git commit бере всі дані, додані в індекс за допомогою git add, і зберігає їх  у внутрішній базі даних, а потім зрушує покажчик поточної гілки на цей зліпок.
6. Для чого використовується  команда init? вмикаеться git для данного репозиторія
7. Що необхідно писати в коментарях комітів? В коментарях комітів потрібно писати усі зміни, для було виконані чи додані
8. Як переглянути список комітів? Щоб подивитися історію коментарів, можна швидко скористатися командою git log
9. Для чого використовується  команда checkout?Команда git checkout дозволяє переміщатися між гілками, створеними командою git branch. При перемиканні гілки відбувається оновлення файлів в робочому каталозі відповідно до версії, що зберігається в цій гілці, а Git починає записувати все ,більше комітів в цій гілці.
